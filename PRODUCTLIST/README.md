## Data Structure and E-Commerce Integration
Each product markdown file contains the following key WooCommerce elements to facilitate ingestion:
- **Title (H1):** Product series and classification.
- **Metadata Fields:** Brand, Category, SKU, and Status.
- **Short Description:** Quick bulleted lists of core parameters.
- **Product Gallery:** Relative links to extracted high-resolution schematics and views.
- **Detailed Description:** Theory of operation, engineering advantages, and standard features.
- **Technical Fact Sheet:** Detailed markdown specification table capturing sizes, pressure ratings, materials, compliance standards, and configurations.
- **References & Sources:** Clear source document citations and website links at the bottom of the page.
